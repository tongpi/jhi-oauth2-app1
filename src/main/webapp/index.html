发生错误 :-(

常见错误原因

  1. 你从IDE启动了应用并且你没有运行 npm start 或 npm run webpack:build.
  2. 运行 npm install时网络出错. 如果您在公司代理之后，则此错误可能是由您的代理引起的。查看Jhipster错误日志，您可能会找到错误的原因。.
  3. JHipster不能工作在你安装的 Node.js版本: 请使用LTS (long-term support) 版本, 因为它是我们唯一支持的版本.

再次构建客户端代码

如果你想快点, 请运行 ./mvnw 构建和运行.

如果希望拥有更多的控制权，以便更容易地调试问题，则应遵循以下步骤:

  1. 使用命令 npm install安装 npm依赖
  2. 使用命令 npm run webpack:build 或 npm start构建客户端代码
  3. 使用命令 ./mvnw 或用 IDE来启动服务
